## 핵심 유저 플로우

### 플로우 1: 오늘의 브리핑 읽기

```
홈페이지 접속 → 뉴스 섹션 스크롤 → 뉴스 상세 클릭 → 원문 링크 이동
```

| 단계 | 테스트 항목 | Pass |
|------|------------|------|
| 1 | 홈페이지 로딩 (3초 이내) | [ ] |
| 2 | 오늘 날짜 표시 확인 | [ ] |
| 3 | 정치/경제/사회 섹션 표시 | [ ] |
| 4 | 뉴스 카드 hover 효과 동작 | [ ] |
| 5 | 뉴스 요약 텍스트 가독성 | [ ] |
| 6 | 원문 링크 클릭 시 새 탭 열림 | [ ] |

### 플로우 2: 아카이브 탐색

```
홈 → "지난 브리핑 보기" → 아카이브 목록 → 날짜 선택 → 이전/다음 네비게이션
```

| 단계 | 테스트 항목 | Pass |
|------|------------|------|
| 1 | "지난 브리핑 보기" 링크 클릭 | [ ] |
| 2 | 아카이브 목록 로딩 | [ ] |
| 3 | 날짜별 브리핑 카운트 표시 | [ ] |
| 4 | 특정 날짜 클릭 → 해당 브리핑 표시 | [ ] |
| 5 | "이전 날" 네비게이션 동작 | [ ] |
| 6 | "다음 날" 네비게이션 동작 | [ ] |
| 7 | "목록" 버튼으로 아카이브 복귀 | [ ] |
| 8 | "오늘로 돌아가기" 동작 | [ ] |

### 플로우 3: 베타 신청

```
홈 → 스크롤 → 베타 CTA → 이메일 입력 → 제출 → 성공 메시지
```

| 단계 | 테스트 항목 | Pass |
|------|------------|------|
| 1 | 베타 신청 CTA 표시 | [ ] |
| 2 | 현재 신청자 수 표시 | [ ] |
| 3 | 이메일 입력 필드 동작 | [ ] |
| 4 | 카카오 ID 입력 필드 (선택) | [ ] |
| 5 | 유효하지 않은 이메일 → 에러 표시 | [ ] |
| 6 | 유효한 이메일 제출 → 로딩 상태 | [ ] |
| 7 | 성공 시 → 감사 메시지 표시 | [ ] |
| 8 | 중복 이메일 → 적절한 에러 메시지 | [ ] |

## 기능별 체크리스트

### 홈페이지 (`/`)

- [ ] 로고 "DAILY WRAP" 표시
- [ ] 서브타이틀 "어제 하루, 한눈에" 표시
- [ ] 오늘 날짜 (한국어 형식) 표시
- [ ] 요일 표시
- [ ] 섹션 인디케이터 (POLITICS, ECONOMY, SOCIETY)
- [ ] Aurora 배경 애니메이션
- [ ] Glass morphism 카드 효과
- [ ] 푸터 저작권 표시

### 뉴스 섹션

- [ ] 섹션 헤더 (한글 + 영문)
- [ ] 섹션 색상 (정치: 빨강, 경제: 초록, 사회: 주황)
- [ ] Featured 뉴스 카드 (전체 폭)
- [ ] Secondary 뉴스 카드 (그리드)
- [ ] 뉴스 제목 표시
- [ ] 뉴스 요약 표시
- [ ] 출처 표시
- [ ] 원문 링크 동작

### 아카이브 목록 (`/archive`)

- [ ] 페이지 제목 "지난 브리핑 아카이브"
- [ ] 브리핑 날짜 목록
- [ ] 각 날짜별 뉴스 개수
- [ ] "오늘" 배지 (해당 시)
- [ ] 목록 아이템 hover 효과
- [ ] "오늘 브리핑으로 돌아가기" 링크

### 아카이브 상세 (`/archive/[date]`)

- [ ] 해당 날짜 표시
- [ ] 이전/다음 네비게이션
- [ ] 목록 링크
- [ ] 뉴스 섹션 표시 (홈과 동일)

### 베타 신청 CTA

- [ ] 장식 테두리 표시
- [ ] 진행률 바 (XX/100)
- [ ] 이메일 입력 필드
- [ ] 카카오 ID 입력 필드
- [ ] 제출 버튼
- [ ] 로딩 상태
- [ ] 성공 상태
- [ ] 에러 상태

## 반응형 테스트

### Desktop (1200px+)

- [ ] 최대 폭 컨테이너 중앙 정렬
- [ ] 2단 뉴스 카드 그리드
- [ ] 큰 타이포그래피 스케일
- [ ] 여유로운 패딩/마진

### Tablet (768px - 1199px)

- [ ] 반응형 패딩 조정
- [ ] 2단 그리드 유지 또는 1단 전환
- [ ] 중간 타이포그래피 스케일
- [ ] 네비게이션 바 가로 배치

### Mobile (< 768px)

- [ ] 단일 컬럼 레이아웃
- [ ] 축소된 타이포그래피
- [ ] 터치 타겟 44x44px 이상
- [ ] 세로 네비게이션 바
- [ ] 읽기 쉬운 본문 크기 (16px+)
