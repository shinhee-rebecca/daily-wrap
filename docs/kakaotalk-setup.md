# 카카오톡 오픈채팅방 설정 가이드

> Daily Wrap 베타 서비스용 카카오톡 오픈채팅방 생성 및 운영 가이드

## 1. 오픈채팅방 생성

### 1.1 기본 설정

1. **카카오톡 앱 실행** > **채팅** > **오픈채팅** 탭 선택
2. **오픈채팅방 만들기** 선택
3. **그룹 채팅** 선택 (1:1 채팅 X)

### 1.2 채팅방 정보 입력

| 항목 | 권장 설정 |
|------|----------|
| 채팅방 이름 | `[Daily Wrap] 오늘의 뉴스 브리핑` |
| 채팅방 설명 | `매일 아침 7시, 어제 하루의 뉴스를 한눈에. Daily Wrap 베타 서비스입니다.` |
| 프로필 사진 | Daily Wrap 로고 이미지 사용 |
| 검색 허용 | OFF (초대 링크로만 입장) |

### 1.3 참여 설정

| 항목 | 권장 설정 |
|------|----------|
| 인원 제한 | 100명 (베타 테스터 수) |
| 참여 코드 | 설정 권장 (무분별 입장 방지) |
| 프로필 공개 | 닉네임만 공개 |

### 1.4 권한 설정

| 항목 | 권장 설정 |
|------|----------|
| 대화 내보내기 | 관리자만 |
| 멤버 초대 | 관리자만 |
| 게시글 작성 | 관리자만 |
| 공지사항 등록 | 관리자만 |

---

## 2. 채팅방 운영

### 2.1 관리자 계정

- **메인 관리자**: Daily Wrap 서비스 계정
- **백업 관리자**: 1명 이상 지정 권장

### 2.2 일일 브리핑 발송 시간

- **발송 시간**: 매일 오전 7:00 KST
- **발송 방법**: 수동 복사/붙여넣기 또는 자동화 스크립트 사용

### 2.3 브리핑 메시지 형식

```

헌법재판소는 19일 오전 10시 대심판정에서 윤석열 대통령 탄핵심판 사건의 5차 변론기일을 열었다.
```

### 2.4 공지사항 활용

채팅방 공지사항에 아래 내용 고정:

```
📢 Daily Wrap 베타 서비스 안내

• 발송 시간: 매일 오전 7시
• 문의: support@dailywrap.kr
• 웹사이트: https://dailywrap.kr

[주의사항]
- 본 채팅방은 뉴스 브리핑 전달 전용입니다
- 개인 대화는 자제해 주세요
- 서비스 관련 피드백은 관리자에게 DM 부탁드립니다
```

## 3. 베타 테스터 관리

### 3.1 초대 프로세스

1. 베타 신청 완료된 사용자에게 이메일로 오픈채팅방 링크 발송
2. 참여 코드 함께 안내
3. 카카오 ID 제공한 사용자는 직접 초대 가능

### 3.2 초대 메시지 템플릿

```
안녕하세요, Daily Wrap입니다.

베타 테스터 신청을 감사드립니다! 🎉

아래 링크로 카카오톡 오픈채팅방에 입장해 주세요.

📱 오픈채팅방 링크: [링크]
🔑 참여 코드: [코드]

입장 후 매일 아침 7시에 뉴스 브리핑을 받아보실 수 있습니다.

궁금한 점이 있으시면 언제든 답장 주세요.

감사합니다,
Daily Wrap 팀
```

### 3.3 퇴장 관리

- 비활성 사용자: 30일 미접속 시 안내 메시지 후 정리
- 위반 사용자: 경고 1회 후 퇴장 조치

## 4. 자동화 연동 (선택)

### 4.1 브리핑 내보내기 스크립트

프로젝트에 포함된 `scripts/export-briefing.ts` 스크립트를 사용하여 카카오톡에 적합한 형식으로 브리핑을 내보낼 수 있습니다.

```bash
# 오늘 브리핑 내보내기
pnpm export-briefing

# 특정 날짜 브리핑 내보내기
pnpm export-briefing 2026-01-19
```

스크립트 실행 후 출력된 텍스트를 복사하여 오픈채팅방에 붙여넣기 합니다.

### 4.2 향후 자동화 계획

- **Phase 6 이후**: 카카오톡 채널 API 연동 검토
- **목표**: 완전 자동 발송 시스템 구축

## 5. 문제 해결

### 5.1 자주 묻는 질문

**Q: 채팅방 인원이 가득 찼습니다.**
A: 인원 제한을 늘리거나, 비활성 사용자를 정리하세요.

**Q: 참여 코드가 유출되었습니다.**
A: 채팅방 설정에서 참여 코드를 변경하세요.

**Q: 스팸 메시지가 올라옵니다.**
A: 관리자만 메시지 작성 가능하도록 권한을 변경하세요.
